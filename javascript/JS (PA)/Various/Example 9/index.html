Application Tab Testing

Open Dev Tools (F12 → Application tab) to inspect browser storage. This demo shows:

  • LocalStorage: Persistent data across sessions
  • SessionStorage: Data persists while tab is open
  • Cookies: HTTP cookies with expiration
  • IndexedDB: Large-scale client-side storage

LocalStorage

Data persists even after closing the browser

SessionStorage

Data persists only while the tab is open

Cookies

Small data files stored on client side

IndexedDB

Large-scale structured data storage with indexing
