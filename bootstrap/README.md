# Bootstrap Framework Learning

Bootstrap CSS framework for rapid development of responsive, mobile-first web pages.

## 📁 Topics

### Grid System

- Container, rows, and columns
- Responsive columns
- Column sizing and offsets
- Auto layout

### Components

- Navigation (navbar, breadcrumb)
- Buttons and button groups
- Forms and form controls
- Cards and media objects
- Alerts and badges
- Modals and tooltips
- Pagination and spinners

### Utilities

- Spacing (margins, padding)
- Display and positioning
- Sizing
- Colors and text utilities
- Flexbox utilities

### Customization

- SASS variables
- Theming
- Custom builds

## 🎯 Key Concepts

- Mobile-first responsive design
- Component-based architecture
- Utility-first approach
- Theme customization
- Accessibility support

## 🛠️ Technologies

- Bootstrap 4/5
- SASS/SCSS
- JavaScript plugins
- npm

## 📖 Contents

Examples demonstrate:

- Grid layout patterns
- Component usage
- Responsive behavior
- Customization
- Best practices

## 🚀 Quick Start

```html
<!-- Include Bootstrap CSS -->
<link
  href="https://cdn.jsdelivr.net/npm/bootstrap@5/dist/css/bootstrap.min.css"
  rel="stylesheet"
/>

<!-- Your content -->

<!-- Include Bootstrap JS -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5/dist/js/bootstrap.bundle.min.js"></script>
```
