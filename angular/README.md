# Angular Framework Learning

Complete Angular framework learning covering components, services, routing, and dependency injection.

## 📁 Topics

### Fundamentals

- Component structure and lifecycle
- Templates and data binding
- Directives (structural and attribute)
- Pipes and transformations
- Services and dependency injection

### Advanced Features

- Routing and navigation
- Lazy loading modules
- Guards (canActivate, canDeactivate)
- Resolvers and interceptors
- HTTP client and API integration

### State Management

- Services for state
- RxJS observables
- Subject and BehaviorSubject
- State patterns

### Forms

- Template-driven forms
- Reactive forms
- Form validation
- Dynamic form generation

### Testing

- Unit tests with Jasmine/Karma
- Component testing
- Service testing
- E2E testing with Cypress/Protractor

## 🎯 Key Concepts

- Component-based architecture
- Unidirectional data flow
- Reactive programming with RxJS
- Dependency injection
- Routing and lazy loading
- Lifecycle hooks

## 🛠️ Technologies

- Angular
- TypeScript
- RxJS
- Node.js / npm
- Angular CLI

## 📖 Learning Path

1. Components and Templates
2. Data Binding and Directives
3. Services and DI
4. Routing
5. Forms (Template and Reactive)
6. HTTP Client and APIs
7. RxJS and Observables
8. Testing

## 🚀 Quick Start

```bash
# Install Angular CLI
npm install -g @angular/cli

# Create new project
ng new my-app

# Start development server
ng serve
```

## 💡 Best Practices

- Smart (container) and dumb (presentational) components
- One responsibility per component
- Use reactive forms for complex forms
- Implement OnPush change detection
- Unsubscribe from observables
- Type everything with TypeScript
