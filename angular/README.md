# Angular Examples

Angular workspace containing demo applications and unit testing examples.

## Contents

- `Demo1/` - Angular demo app 1
- `Demo2/` - Angular demo app 2
- `Demo3/` - Angular demo app 3
- `Unit-Testing/` - Angular unit testing examples

## Notes

Each subfolder is an Angular CLI app. Run `npm install` and `ng serve` inside the chosen folder to launch the demo.
