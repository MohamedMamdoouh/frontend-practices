# TypeScript Learning & Practice

Type-safe JavaScript development with TypeScript, providing compile-time type checking and modern language features.

- Arrays and tuples
- Union and intersection types
- Type aliases
- Enums

### Advanced Types

- Generics
- Conditional types
- Type guards and assertions
- Utility types
- Type inference

### Object-Oriented Programming

- Classes and inheritance
- Interfaces and abstract classes
- Access modifiers (public, private, protected)
- Getters and setters
- Static members

### Functional Programming

- Higher-order functions
- Function overloading
- Decorators
- Modules and namespaces

### Type Definitions

- Type declarations
- Declaration files (.d.ts)
- Third-party type definitions (@types)
- Creating type definitions

## 🚀 Getting Started

```bash
# Install TypeScript
npm install -g typescript

# Compile TypeScript
tsc filename.ts

# Watch mode
tsc --watch
```

## 🔧 Configuration

- **tsconfig.json** - Compiler configuration
- **Type checking strictness** - Configure strict mode
- **Module system** - CommonJS vs ES modules
